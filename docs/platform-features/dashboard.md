# Dashboard Overview

Manage your Pixel Streaming projects, monitor usage, and track performance from the Third Space dashboard.

## Accessing the Dashboard

Log in at [app.thirdspaceinteractive.ca](https://app.thirdspaceinteractive.ca)

Your dashboard is the central hub for all Third Space operations.

## Projects View

### Project Cards

Each deployed project shows:
- **Project name** and description
- **Status indicator** (Ready, Deploying, Stopped, Error)
- **Live URL** and copy button
- **Quick actions** (Launch, Settings, Delete)
- **Last deployed** timestamp
- **Current usage** (active connections)

### Project Actions

**Launch Stream:**
- Opens stream in new tab
- Test before sharing with users
- Verify functionality

**Settings:**
- Update project name/description
- Configure custom domain
- Set password protection (Pro+)
- Adjust idle timeout (Enterprise)

**Analytics:**
- View session history
- Credit consumption over time
- User geographic distribution

**Delete:**
- Remove project and free storage
- Cannot be undone

## Create New Project

1. Click **+ New Project**
2. Enter project details
3. Upload packaged build (.zip)
4. Monitor deployment progress
5. Receive streaming URL when ready

## Credit Management

### Current Balance

Dashboard shows:
- **Credits remaining** this billing cycle
- **Credits consumed** today/this week/this month
- **Projected usage** at current rate
- **Days until reset** (monthly billing)

### Usage Alerts

Set up notifications:
- Email at 80% depletion
- SMS at 90% (Pro/Enterprise)
- Auto-top-up triggers

### Purchase Credits

**Quick top-up:**
1. Click **Buy Credits**
2. Select pack size
3. Payment via Stripe
4. Credits available instantly

## Real-Time Monitoring

### Active Sessions

See currently connected users:
- Number of active connections
- Session duration
- Credits consumed this session
- Geographic location (Pro+)

### Performance Metrics

Monitor stream health:
- **FPS** (frames per second)
- **Bitrate** (Mbps)
- **Latency** (ms)
- **Connection quality**

## Analytics (Pro/Enterprise)

### Session History

Track all past sessions:
- Start/end times
- Duration
- Credits consumed
- User location
- Device type (desktop/mobile)

### Usage Trends

Visualize patterns:
- Daily/weekly/monthly graphs
- Peak usage times
- Per-project breakdown
- Cost projections

### Export Data

Download usage reports:
- CSV export for accounting
- Custom date ranges
- Per-project or account-wide

## Account Settings

### Profile

- Update email and password
- Manage billing information
- Set notification preferences

### Billing

- View invoices and payment history
- Update payment method
- Switch subscription tier
- Download receipts

### Team Management (Enterprise)

- Invite team members
- Set role-based permissions
- Manage API keys
- Audit logs
